# My Personal Website | Hosted on GITHUB

### FYI: My real website is www.ayushisingh.com

## Overview

This project contains the source code for my personal portfolio website.
The website is built using simple front-end technologies and is hosted using GitHub Pages.

The purpose of this project is to showcase:

* My professional profile
* My projects
* Media and speaking engagements
* Contact information
* Technical interests and learning journey

This repository focuses only on the **website code and content**.
A separate repository will be used to demonstrate **DevOps automation and cloud deployment for the same website**.
